選擇器寫法

寫法差異說明 ▸

一般 — 直接加在 class=""，永遠生效。
<span class="bg-duck-primary">

hover: — 滑鼠移入時觸發。
<span class="hover:opacity-80">

focus: — 鍵盤 Tab 或點擊輸入框聚焦時觸發。

active: — 滑鼠按下未放開時觸發。

disabled: — 元素帶有 disabled 屬性時觸發。

dark: — 系統或手動切換到深色模式時觸發。

group- / peer- — 需父層 / 前一兄弟元素分別帶有 class="group" / class="peer"，才能在對應互動時觸發。
<div class="group"><span class="group-hover:text-white">

aria- — 元素帶有對應 aria-* 屬性且符合條件時觸發。

responsive — sm: / md: / lg: 等斷點 variant，超過指定螢幕寬度才生效。
